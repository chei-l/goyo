Create a landing page in `content/_index.md` with `landing.html` template.

The landing page template provides several customizable sections to build an engaging homepage for your documentation site. Each section is optional and can be configured independently.

## Available Sections

- **[Overview](./overview/)**: Learn about section ordering and basic landing page setup
- **[Hero Section](./hero/)**: Main banner with title, description, background image, and CTA buttons
- **[Features Section](./features/)**: Features grid with title, description, and icons
- **[Trust Section](./trust/)**: Display logos of companies or projects that trust you
- **[Easy Command Section](./easy_command/)**: Tabbed commands with optional link tabs
- **[Showcase Section](./showcase/)**: Tabbed content with images or text
- **[Social Proof Section](./social_proof/)**: User testimonials
- **[Final CTA Section](./final_cta/)**: Final call-to-action with optional image
