`content/_index.md`에서 `landing.html` 템플릿으로 랜딩 페이지를 생성합니다.

랜딩 페이지 템플릿은 문서 사이트의 매력적인 홈페이지를 구축할 수 있도록 여러 커스터마이징 가능한 섹션을 제공합니다. 각 섹션은 선택사항이며 독립적으로 구성할 수 있습니다.

## 사용 가능한 섹션

- **[개요](./overview/)**: 섹션 순서 및 기본 랜딩 페이지 설정에 대해 알아보기
- **[Hero 섹션](./hero/)**: 제목, 설명, 배경 이미지, CTA 버튼이 있는 메인 배너
- **[Features 섹션](./features/)**: 제목, 설명, 아이콘이 있는 기능 그리드
- **[Trust 섹션](./trust/)**: 당신을 신뢰하는 회사나 프로젝트의 로고 표시
- **[Easy Command 섹션](./easy_command/)**: 링크 탭을 지원하는 탭 형태의 명령어
- **[Showcase 섹션](./showcase/)**: 이미지 또는 텍스트가 있는 탭 형태의 콘텐츠
- **[Social Proof 섹션](./social_proof/)**: 사용자 추천사
- **[Final CTA 섹션](./final_cta/)**: 선택적 이미지가 있는 최종 행동 유도
